# Backend

FastAPI + SQLite + LanceDB. Python 3.11+.

## Dependency Management

Three source files drive dependency installation:

- **`backend/requirements.txt`** — Production runtime dependencies (includes `lancedb`, `pyarrow`, `unstructured`, `sentence-transformers`).
- **`backend/requirements-ci.txt`** — CI runtime dependencies; excludes the heavy packages stubbed by tests to avoid multi-GB installs.
- **`backend/requirements-dev.txt`** — Development and test tools (`ruff`, `pytest` plugins, etc.).

The generated lockfiles **`backend/requirements-lock.txt`** and **`backend/requirements-lock-ci.txt`** are kept in sync with the source files above. Do not edit lockfiles directly — they are regenerated by the update scripts.

## Regenerating Lockfiles

Run the update script from the repo root:

- Linux/macOS: `scripts/update-lock.sh`
- Windows: `scripts/update-lock.ps1`

Both scripts require `pip-tools`. If `pip-compile` is not available, the scripts will install it automatically before generating the lockfiles.

## Installing Dependencies

For production use:

```
pip install -r backend/requirements-lock.txt
```

For CI use:

```
pip install -r backend/requirements-lock-ci.txt -r backend/requirements-dev.txt
```

## Running Tests

```
pytest tests/
```

Run from the `backend/` directory. The project uses Python 3.11 in CI.
